OpenGL Examples  
----------------
Simple OpenGL codes using the OpenGL Library

* Compile and Execute

```
$ g++ <filename.cpp> -lGL -lglut -lm -o <outputfile>
$ ./<outputfile>  
```

* Dependencies
  * freeglut3
  * freeglut3-dbg
  * freeglut3-dev

* To install the dependencies

```
$ sudo apt-get install freeglut3 freeglut3-dbg freeglut3-dev
```
